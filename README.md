# 🧠 Cerebrum SMP

Platform belajar Matematika SMP interaktif dan **100% gratis** — seperti Duolingo tapi untuk Matematika!

## Fitur
- Gamifikasi: XP, level, streak, combo, hearts
- Konten live dari Google Sheets (tanpa redeploy)
- Kurikulum Merdeka — Matematika Kelas 7

## Setup
1. Clone repo ini
2. Edit `public/config.js` dengan Google Sheet ID kamu
3. Deploy ke Vercel

## Tech Stack
- Vanilla HTML/CSS/JS (zero framework)
- PapaParse untuk parsing CSV
- Google Sheets sebagai CMS
- Vercel untuk hosting
